De Website

Omschrijving

Aan deze website zijn we redelijk vroeg begonnen om een goed resultaat te krijgen. Toch waren er veel uitdagingen in het proces van de opmaak van deze website. Om de website er goed uit te laten zien, heb je veel referenties en code nodig. Niet alleen CSS code (de achterliggende code achter de kleuren en ligging van objecten) is gebruikt. Ook javascript (Voor animaties) en php (Het versturen van mails. Dit is nodig in de contact sectie onderaan op de homepagina). Het implementeren van foto's, hyperlinks en video's op een acceptabele manier is ook een uitdaging. Om deze site in orde te brengen heb ik Visual Studio Code gebruikt, naar mijn mening de beste aanpasbare en makkelijke editor voor code die er bestaat.

Ik heb gebruik gemaakt van het internet, zoals de meeste andere leerlingen in mijn klas voor inspiratie en werkende code te implementeren. De rechten hiervan zijn te vinden in de HTML code van mijn project, maar ook onderaan op elke pagina in deze website.
